Data Flow
Props & 数据流

🌐 → 📦 chrome.tabs → useTabs

Hook 监听三个 chrome 事件，防抖后刷新：

→onCreated立即刷新
→onRemoved立即刷新
→onUpdated200ms 防抖刷新

📦 → 🎯 useTabs → Dashboard

返回两个值供 Dashboard 消费：

↓groupsMap<domain, TabInfo[]>
↓allTabsTabInfo[] 所有 tab 原始列表

🎯 → 🗂️ Dashboard → DomainCard

↓domainstring — 域名标题
↓tabsTabInfo[] 去重后的列表
↓onCloseTab(tabId) → chrome.tabs.remove
↓onShake() → contentRef 屏幕震动

🗂️ → 💣 DomainCard → TabItem

↓tabTabInfo 单条 tab 数据
↓onClose(tabId) 关闭回调
↓forceShowClose长按时强制显示 ✕
↓isCharging显示进度环
↑onLongPressStart/Cancel上报长按状态
↑ref (TabItemHandle)triggerBomb / triggerBombVisual

🎯 → 🔍 Dashboard → SearchBar

↓valuestring 受控值
↓onChangesetSearchQuery
↓onClear() → setSearchQuery("")

🎨 tokens.ts — 全局共享

所有组件 import 同一份设计 token，保证视觉一致性：

DomainCard.tsx
handleClearDuplicates 执行：
1. 先查当前活跃 tab（chrome.tabs.query({active:true})）
2. safeSlice(ids, url)：若当前 tab 在这组 → 保留它；若 URL 相同（TabBoom 自身）→ 关全部；否则 → ids.slice(1)
